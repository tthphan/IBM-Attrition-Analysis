# IBM-Attrition-Analysis
Subject People Analytics: Strategy and Practice.

## Project Overview
This project analyses the IBM HR Analytics dataset (1,470 employees) to identify the root causes of attrition and develop data-driven retention strategies. The goal was to build a predictive model to forecast future turnover and provide actionable insights for management.

## Tech Stack
- Data Analysis: R, Microsoft Excel

- Visualisation: R, SAS Viya, Microsoft Excel

- Machine Learning: Logistic Regression (LR), k-Nearest Neighbor (KNN), Extreme Gradient Boosting (XGB)

## Key Features
- Trend Analysis: Utilised SAS Viya and Excel to visualise dataset distributions and identify emerging turnover trends.

- Talent Identification: Developed a Key Employee Index and a dedicated Dashboard to monitor high-value employees who are critical to the organisation.

- Model Selection: Conducted iterative Logistic Regression modeling, using ANOVA to compare variable sets.

- Benchmarking: Evaluated performance across LR, KNN, and XGB models. Logistic Regression was selected as the champion model for its superior accuracy and interpretability.

- Predictive Forecasting: Applied the final model to generate attrition predictions for the upcoming year.

## Strategic Recommendations
Based on the model's findings, I proposed an example of retention interventions:

- Increasing Training Opportunities: Data suggested a significant correlation between professional development and employee retention.
